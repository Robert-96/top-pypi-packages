# top-pypi-packages

A web page with the most-downloaded packages from PyPI.

https://robert-96.github.io/top-pypi-packages/30-days.html

## Build

```
$ pip install -r requirements.txt
$ python -m script
```

## Test

```
$ pip install -r requirements-dev.txt
$ pytest tests
```
